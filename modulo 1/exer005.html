Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Também podemos usar a tag semântica address

Ela é usada para mostrar endereço, por isso pula a linha

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado no final.

A gente define a cor dessa marcação com css ou em um arquivo próprio de css ou na tag style no próprio html

Texto grande e pequeno

Estamos criando um texto grande (Tag big, não-semântica) e um texto pequeno nesse parágrafo. (Tag small, semântica)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Tag del, semântica

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usamos a tag ins que é semântica.

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, Tag sup semântica

Texto subscrito

Para inserir coisas do tipo H2O, Tag sub semântica

Códigos

Usamos o code para mostrar códigos em seu site


        x = 2
        if x == 2:
            print('O número é 2')
        else:
            print('O número não é 2')

Usamos outra tag, o pre para formatar o código e mostrar do jeito que foi digitado.

Citações

Para mostrar citações, usamos o A tag q, que vai colocar automaticamente as aspas, mais usada no meio de parágrafos

Para citações longas usamos mais o blockquote, por ter recuo e também podemos colocar o link da citação original no parâmetro cite.

Textos Invertidos!

Para inverter textos que nem neste titulo usamos a tag bdo, para tomar forma, temos que definir o parâmetro dir. Temos duas dir.
RTL: Direita para esquerda (right-to-left)
LTR: Esquerda para direita (left-to-right)
Assim definimos como o texto vai se comportar
