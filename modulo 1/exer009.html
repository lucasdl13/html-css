CSS Inline

Temos diferentes tipos de usos de CSS, o Inline é usado em casos específicos quando você quer uma cor especial para determinado texto

Esse parágrafo está com cor diferente dos demais

Ela é usada diretamente na própria tag de HTML com o atributo style, por essa mesma razão, que ele não é recomendado.

CSS Interno

Esse é usado com a tag style no head da estrutura, normalmente usado para pequenas configurações. Neste caso, está deixando os títulos verdes

CSS Externo

Como o nome sugere, o CSS Externo usa arquivos de fora, com a tag link:css no header, fará a lincagem com um arquivo CSS (o próprio VSC pode criar o arquivo para você). Ele está fazendo toda a estrutura CSS desse site, então assim, o CSS Externo é usado para configurações complexas para deixar o código mais organizado, além de que o mesmo arquivo css pode ser usado para diversas páginas, facilitando as mudanças do site

Observação: Por precaução, colocar na primeira linha do arquivo CSS: "@charset "UTF-8";" para compatibilidade com o arquivo HTML
